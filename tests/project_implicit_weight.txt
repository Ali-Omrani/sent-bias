# The Project Implicit IAT only included terms for "Good" and "Bad;"
# fat and thin people were represented by images of figures.  The terms
# for fat and thin people are an ad-hoc set of adjectives produced by
# starting with "fat" and using thesaurus.com to generate synonyms and
# antonyms.
FatTerms:fat,overweight,heavy,thick,plump,obese,chubby,large,wide,hefty
ThinTerms:thin,underweight,light,skinny,bony,gaunt,slender,scrawny,lanky,lean
Good:fabulous,love,enjoy,adore,excitement,delight,friend,smiling
Bad:awful,gross,horrific,poison,yucky,detest,hate,sadness
